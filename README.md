Human emotion recognition plays a crucial role in improving human
computer interaction systems. The ability of machines to understand human 
emotions can significantly enhance applications in areas such as healthcare, 
education, security, and artificial intelligence. This project presents a deep 
learning-based system that detects human emotions from facial expressions in 
real time using webcam input. 
The proposed system utilizes a hybrid deep learning model combining 
ResNet-50 and EfficientNet-B3 architectures to improve feature extraction and 
classification accuracy. Facial images are captured using a webcam and 
processed using OpenCV for face detection. The detected face region is 
preprocessed and fed into the hybrid convolutional neural network to extract 
deep facial features. These features are then fused and passed to a fully 
connected layer for emotion classification. 
The system is trained and evaluated using the RAF-DB dataset, which 
contains approximately 15,000 real-world facial images categorized into seven 
emotional classes: Angry, Disgust, Fear, Happy, Neutral, Sad, and Surprise. 
Experimental results show that the model achieves a demonstrating its 
effectiveness in recognizing emotions under different lighting conditions and 
facial poses. 
The developed system performs emotion detection in real time and can be 
used in applications such as intelligent tutoring systems, mental health 
monitoring, and emotion- aware AI systems.
